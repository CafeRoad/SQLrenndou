#### 솔데스크 801호. 팀 프로젝트입니다.   
 맛집 추천 사이트는 어디서든 많이 찾아볼 수 있지만 카페들의 정보들만 모아볼 수 있는 사이트는 찾아볼 수 없어서 따라서 유저들의 솔직하고 거짓 없는 리뷰로 카페 맛집을 선택할 수 있고 그 카페들의 정보들을 쉽고 빠르게 검색할 수 있는 사이트를 제작하게 되었습니다.
